## General remarks 
This toolkit covers the algorithm by [Ahlfeldt, Heblich, Seidel, Yin (2026)](https://github.com/Ahlfeldt/DPs/blob/main/GA_SH_TS_FY_-_Productivity.pdf) to generate micro-geographic commercial rent indices indieces using the RWI-GEO-REDC (v4) dataset.

The two subfolders contain the indices at postcode level (PLZ) and loal labor market level (LLM). We also report the CBD rent for each local labor market in the LLM level dataset.
