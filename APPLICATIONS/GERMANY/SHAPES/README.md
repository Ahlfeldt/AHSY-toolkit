# General remarks
This folder contains the shape files for Germany at postcode level and local labor market (LLM) level.

We use the 2017 version of local labor market defined by BSSR (Bundesamt für Bauwesen und Raumordnung).

These local labour markets contain multiple postcode regions and are defined such that at least 65\% of workers commute within their own LLM and commute times do not exceed 45 minutes (one way). A short description on how LLMs are delineated can be found [here](https://www.bbsr.bund.de/BBSR/DE/forschung/raumbeobachtung/Raumabgrenzungen/deutschland/regionen/arbeitsmarktregionen/Arbeitsmarktregionen.html) (in German).
